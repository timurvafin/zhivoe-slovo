# Живое слово

Скилл для очеловечивания русскоязычного текста со следами LLM. Возвращает в текст автора — мнение, ритм, характер — вместо того чтобы просто прогнать его через список запретов.

Имя — отсылка к «Слову живому и мёртвому» Норы Галь, главной русской книге о борьбе с канцеляритом. Подход редакторский: текст делается живым **для читателя**, а не для AI-детектора. Никаких анти-детекторных трюков (намеренных опечаток, поломанной типографики) — только честная редактура.

Это синтез лучшего из семи открытых MIT-скиллов (русских и английских первоисточников), собранный в одну непротиворечивую систему. Полный список источников — в [ATTRIBUTION.md](ATTRIBUTION.md).

## Чем отличается

- **Не навреди.** Главный принцип — «лучше пропустить машинный текст, чем испортить живой». Перед правкой скилл сверяется с каталогом ложных срабатываний и признаков живого письма. Чистый, формальный или «суховатый» текст — это не ИИ.
- **Тире судим, а не казним.** Большинство деслоп-скиллов банят длинное тире наглухо — для русского это калечит интонацию. Здесь тире оценивается по плотности (≤3 на 1000 знаков), жанру и голосу автора, а не запрещается.
- **Голос автора.** Голосовой паспорт `.humanizer/voice.json` хранит, как пишет конкретный человек, и переживает сессии. Жанровые пресеты под Telegram, Habr, vc, email, документацию.
- **Детерминированные маркеры.** Regex на чат-бот-артефакты (`:contentReference`, `utm_source=chatgpt.com`, `【N†source】`) — единственные стопроцентные приметы ИИ.
- **Русская физика.** Канцелярит, кальки с английского, «является», цепочки родительного, морфология, частицы, идиомы — то, чего нет в англоязычных скиллах.

## Установка

Как локальный скилл проекта — положить папку в `.claude/skills/zhivoe-slovo/`. Вызов: `/zhivoe-slovo` или фразой «очеловечь текст», «убери следы нейросети».

## Использование

```
/zhivoe-slovo
[вставить текст]
```

Режимы:
- **полное редактирование** (по умолчанию) — переписать;
- **аудит** («проверь», «что выдаёт?») — только диагностика + структурные метрики, без правок;
- **точечно** («убери только канцелярит») — одна категория.

Для лучшего результата дай образец своего письма (2-3 абзаца) — скилл откалибрует паспорт под твой голос и сохранит его в `.humanizer/voice.json`. Шаблон паспорта со всеми полями — в `examples/voice.template.json`.

## Где он реально нужен

На **AI-черновиках перед публикацией** (особенно лонгриды, где модель скатывается в канцелярит). На **уже живом авторском тексте** скилл честно скажет «зелёное, не трогаю» — и это правильное поведение, а не отказ работать.
